4. 종합 결론

• 부리 길이: Chinstrap 펭귄 > Gentoo 펭귄 > Adelie 펭귄
• 부리 깊이: Adelie ≈ Chinstrap > Gentoo
• 날개 길이: Gentoo > Chinstrap > Adelie
• 체중: Gentoo (가장 무거움) > Chinstrap ≈ Adelie
• 개체 수: Biscoe 섬에 Gentoo 펭귄이 가장 많고, Dream 섬에는 Chinstrap 펭귄만 존재.
